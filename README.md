# Student Registration and Placement Management Form

This project simulates a real-world student placement registration system using structured HTML forms.

## Project Overview
The application captures complete student information required for placement processes including:

- Personal and demographic details
- Contact information
- Academic background
- Technical skills
- Domain preferences
- Salary expectations
- Account creation and consent agreement

## Key Highlights
- Designed multi-section form layout
- Implemented various input types (text, date, radio, checkbox, dropdown)
- Structured form logically for usability
- Deployed successfully using GitHub Pages

## Technologies Used
- HTML5
- Semantic form elements

## Learning Outcome
This project strengthened my understanding of frontend structure design and real-world data collection systems commonly used in placement portals.
